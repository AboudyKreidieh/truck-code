Detailed Description

j1939_interpreters.h

This file contains the abstract class for J1939 message processing. The abstract class consists of methods for:

  • processing: converting a pdu variable into its message-specific type
  • comparing: checking if a pdu message is the same type as the one covered by any specific subclass of the abstract class
  • printing: printing/logging the information in a message
  • publishing: sending messages to a pub/sub servers
  • importing: collecting data from a file of numeric formatted data and placing it in a message-specific object

This file contains also interpreters for all used J1939 messages.
