Detailed Description

can_man.c

Resource manager for the SSV CAN board.

To use both 82527 controllers of the SSV board, two instances of this process must be executed.

Digital I/O on the board is also supported by the same devctls used by PATH DAS (Data Acquisition System) drivers.
