Detailed Description

can.h

This file contains the class needed to read and write messages to the Control Area Network (CAN) card.

Messages during the reading and writing methods are processed from or placed in a can_message object, also located in this folder.

Finally, this file also contains a method for obtaining a pointer to an active CAN class, or creating a new one. This methods is used by the JBusSender and JBusReceiver classes in order for all other processes to share the same CAN class.
