Detailed Description

can_dev.cpp

Device level code, based on sample code provided by SSV, originally ported to QNX4 by Jorge M. Estrela da Silva, 2002 ISEP, Porto, Portugal.

Filtering into different message registers is not implemented. All transmit is done from MSG1, all received in MSG15.
